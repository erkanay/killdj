# killdj

killdj is a simple tool to terminate ports running by django.

#### usage 
```bash
...$ sudo ln -s /full/path/to/your/killdj.sh /usr/local/bin/killdj
...$ chmod +x killdj.sh
```
```bash
...$ killdj
```
